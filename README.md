# Hacking With SwiftUI - Bucket List

## Overview/Description
An app that builds a private list of places on a map with a name, description and a list of places that are nearby

## What I Learned
- Using 'documentsDirectory', it can be used as a file location to hold data.
- Using Comparable to create a custom compariable function.
- Effectively using Enums, in this app it was used on the EditView to show a list of nearby places and depending on the response from the URLSession call, it would return a particular view.
- Introduction of MapKit and displaying different MapStyles.
- Creating custom annotations for MapKit.
- Working with different tap gestures. In this app I'm working with a single tap gesture and a long tap gesture.
- Introduction of FaceID and TouchID and biometric authorisation.
- Converting CGPoint to CLLocationCoordinate2D.
- Downloading data from Wikipedia and providing a list of locations. To do this, I used URL and URLSession.
- Introduction to the MVVM (Model View View-Model) design pattern.
